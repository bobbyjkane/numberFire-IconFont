# numberFire-IconFont-006
All icons for numberfire.com's redesign.

Refer to demo.

## Usage

### HTML
```
<span class="nf-icon icon-*"></span>
```

### Basic Styling

```
.nf-icon {
  font-family: 'numberfire-iconfont' !important;
  speak: none;
  font-style: normal;
  font-weight: normal;
  font-variant: normal;
  text-transform: none;
  color: #454545;
  line-height: 1;
  top: 2px;
  font-size: inherit;
  text-rendering: auto;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}
```
